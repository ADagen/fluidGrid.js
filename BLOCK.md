Класс Block
============

В коллбеке метода плагина 'addBlock' параметром возвращается новый блок,
дающий интерфейс для взимодействия (для вызова в используемых виджетах).

block.$html - базовый html-элемент блока, используется для встраивания виджета

block.$placeholder - используется при перетаскивании блока, показывает текущую проекцию блока на сетку ячеек

block.switchSize - принимает два агрумента типа int (новые colspan и rowspan) и переключает блок в новый размер.
